como dice wiki...:

El HTML es un lenguaje de marcas para la creación de páginas web. Fue creado por Tim Berners-Lee en 1991. HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). El HTML se utiliza para estructurar el contenido en la web, como texto, imágenes, enlaces, formularios y otros elementos multimedia. Los documentos HTML están compuestos por una serie de elementos que se representan mediante etiquetas. Estas etiquetas indican al navegador cómo debe mostrar el contenido. El HTML es un estándar abierto mantenido por el World Wide Web Consortium (W3C).
